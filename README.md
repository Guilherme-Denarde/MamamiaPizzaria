
# Mammamia Pizzaria Project Documentation

## Overview

Mammamia Pizzaria is a full-stack web application designed to simulate an online food ordering platform, similar to iFood. It consists of a backend developed in Java 19 and a frontend crafted using Angular 16.

## Repository Structure

- **Backend**: Located in the `backend` folder, it contains all the Java code and resources necessary for the server-side functionality.
- **Frontend**: The `frontend` folder houses the Angular 16 application, responsible for the user interface and client-side interactions.

## Backend (Java 19)

### Setup and Installation

1. **Prerequisites**: Ensure Java 19 is installed on your system.
2. **Dependencies**: All dependencies are listed in the `pom.xml` file and can be installed via Maven.
3. **Running the Application**: Use the command `java -jar <backend-jar-file>` to start the server.

## Frontend (Angular 16)

### Setup and Installation

1. **Prerequisites**: Node.js and Angular CLI must be installed.
2. **Dependencies**: Run `npm install` in the `frontend` directory to install required packages.
3. **Running the Application**: Execute `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
